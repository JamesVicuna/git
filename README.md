** GIT Workflow

** git init 
- initializes git on current directory

* git add file-name
- stages changes for commit

* git commit -m "message-here"
- saves add changes to current local branch. 

* git push 
- pushes / merges changes to remote git repository
- if changes are made on main branch pushes directly to main
- if changes are made on a new branch, a pull request is opened before merging changes to main
